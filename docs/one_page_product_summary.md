# ProofPay AI
## AI-verified payment requests for student vendors

---

### The Problem
Nigerian students lose money daily to fake vendors on WhatsApp and Instagram. No trust signal. No accountability. No record.

### The Solution
ProofPay AI lets buyers see an AI trust score before every Kora-powered payment. Sellers create verified payment requests. Buyers see the verdict. Kora confirms the payment.

### How It Works
1. Seller creates payment request in under 60 seconds
2. AI scores the vendor: profile, history, disputes, and amount
3. Buyer sees trust verdict before paying
4. Buyer pays through Kora Checkout
5. Kora webhook confirms payment automatically
6. Both parties have a verified transaction record

### Trust Score Factors
- Profile completeness
- Transaction history
- Dispute rate
- Account name consistency
- Amount reasonableness
- Anomaly detection

### Business Model
- 1% transaction fee per verified payment
- NGN 3,000/month vendor subscription
- B2B Trust Score API for marketplaces
